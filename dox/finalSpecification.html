Final Specification for Radard

Introduction

The Rardard social network web application is a web-based chat tool developed for use within education to allow students to speak to other students within a given proximity. The web based application permits the user to select an area unto which they are located and provides the ability to chat within a window to other users that are also situated in proximity to that location.

Objectives

The objective of this web application was to create a new social network that allowed users to select a location and chat to other users using the service within the local environment. A secondary objective of this web application was to permit lecturers and students to communicate with each other within a class environment using this service. Allowing the student to ask questions that they may not feel able to ask.

Key Audiences

The initial prospective audience for this web application were students and lecturers within the education sector. Other secondary key audiences include: Museum goers who can communicate with others and guides, Employees within a workplace etc.

Rationale

The approach for this web application was to focus on the functionality of the main feature for the application and once this function (instant messenger) has been correctly and proficiently coded, the other features which added additional functionality to the application would be added as it progressed. Therefore as the web application currently stands the main feature of the instant messenger is working correctly to include creation of the database and the use of AJAX and json and PDO to communicate sufficiently with it. This allows the user to enter into the chat and instantly send and receive messages from others within that chat messenger.
